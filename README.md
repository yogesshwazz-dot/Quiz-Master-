# ❓ Quiz Master

A simple and interactive quiz web application built with **HTML**, **CSS**, and **Vanilla JavaScript**.

## 📌 About

This project was created to practice JavaScript by building a fun quiz application. Users can answer multiple-choice questions, see their score, and restart the quiz anytime.

## ✨ Features

* Multiple-choice questions
* Instant feedback
* Score tracking
* Restart quiz
* Responsive design

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
Quiz-Master/
│── index.html
│── style.css
└── script.js
```

## 🚀 How to Run

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Start playing the quiz!
